Manual Installation Steps

  1. Download the extension files from GitHub
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" in the top right corner
  4. Click "Load unpacked" and select the extension folder
  5. Pin the extension to your toolbar for easy access

Quick Start Guide

1. Access the Extension

Click the A/B Test Calculator icon in your browser toolbar. If you don't see it, click the puzzle piece icon and pin it.

2. Validate Your Experiment Setup

  • Hypothesis: Describe what you're changing and expected impact
  • Baseline Rate: Current conversion rate (e.g., 5%)
  • Minimum Detectable Effect: Smallest change you want to detect (e.g., 0.5%)
  • Traffic: Daily visitors and allocation percentage

3. Get Instant Results

The extension calculates:

  • Required sample size per variant
  • Estimated test duration
  • Statistical power analysis
  • Trade-off scenarios

4. Analyze Your Results

Once your test is complete, use the "Analyze Results" tab to:

  • Test statistical significance
  • Calculate confidence intervals
  • Get AI-powered interpretation (with API key)

Features & Capabilities

Statistical Calculations

  • Sample Size Calculator: Determines required participants per variant
  • Power Analysis: Ensures your test can detect meaningful changes
  • Duration Estimation: Predicts how long your test needs to run
  • Significance Testing: Two-proportion z-test for result analysis
  • Confidence Intervals: 95% confidence bounds for your results
  • Effect Size: Relative and absolute lift calculations

AI-Powered Insights (Optional)

  • Hypothesis Analysis: AI evaluates your experiment design
  • Results Interpretation: Plain-English explanations of statistical findings
  • Recommendations: Next steps based on your results
  • Risk Assessment: Identifies potential issues with your experiment

Data Management

  • Local Storage: All data saved securely in your browser
  • Export Options: Download results as CSV or JSON
  • Form Memory: Automatically saves your inputs
  • Settings Backup: Export/import your configuration

AI Setup (Optional)

Note: AI features are completely optional. The extension works perfectly for statistical calculations without any API keys.

Google Gemini (Recommended)

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key" and copy it
  4. Open A/B Test Calculator → Settings
  5. Paste your key in the Gemini API field
  6. Click "Test Connection" to verify

Anthropic Claude

  1. Visit Anthropic Console
  2. Create an account or sign in
  3. Navigate to API Keys and create a new key
  4. Copy the key (it starts with "sk-ant-")
  5. Open A/B Test Calculator → Settings
  6. Paste your key in the Claude API field
  7. Click "Test Connection" to verify
Security: Your API keys are stored locally in Chrome's encrypted storage and never sent to our servers. They're only used for direct communication with AI services.

Frequently Asked Questions

Do I need an internet connection to use the extension?
No! All statistical calculations run locally in your browser. You only need internet for optional AI insights.
How accurate are the statistical calculations?
Very accurate. Our calculations use the same formulas as industry-standard tools like SciPy and match server-side implementations within numerical precision limits.
Is my data secure?
Yes. All data is stored locally in your browser using Chrome's secure storage API. We don't collect, store, or transmit any of your experiment data.
What's the difference between this and online calculators?
Speed, privacy, and AI insights. No waiting for page loads, no data sent to servers, and optional AI analysis of your experiments.
Can I use this for non-web experiments?
Absolutely! The statistical methods work for any A/B test - mobile apps, email campaigns, offline experiments, etc.
Why do I need to provide my own AI API keys?
This ensures your data privacy and gives you control over AI costs. We don't have access to your experiment data or AI conversations.
Does this work in other browsers?
It works in most Chromium-based browsers (Edge, Brave, Opera) but is optimized for Chrome. Firefox support may come in the future.
How do I export my results?
After running calculations, look for the export buttons. You can download results as CSV (for spreadsheets) or JSON (for other tools).

Troubleshooting

Extension Not Loading

  • Refresh the extension at chrome://extensions/
  • Make sure Chrome is up to date (version 88+)
  • Try disabling and re-enabling the extension

AI Features Not Working

  • Verify your API key is correct and has credits
  • Use "Test Connection" in settings to diagnose issues
  • Check if your API key has the necessary permissions
  • Try switching between Gemini and Claude if one isn't working

Calculations Seem Wrong

  • Double-check your input values (percentages should be decimals like 0.05 for 5%)
  • Ensure your baseline rate is realistic for your context
  • Consider if your MDE is too small to detect reliably

Performance Issues

  • Clear the extension's data in settings if it becomes slow
  • Restart Chrome if the extension becomes unresponsive
  • Check if other extensions are conflicting
