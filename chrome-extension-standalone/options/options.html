ℹ️ About PM Tools - Standalone Edition

Self-sufficient A/B testing tool with bring-your-own-key AI integration

🚀 What Makes This Special

  • 🔑 Bring Your Own Keys: Use your own AI API keys for complete control
  • 🏠 No Backend Required: All calculations run locally in your browser
  • 🔒 Privacy First: Your data never leaves your browser
  • 💰 Cost Control: You manage your own AI usage and costs
  • ⚡ Fast: No network round-trips for statistical calculations

🎯 Core Features

  • Local statistical calculations (sample size, power analysis, significance testing)
  • AI-powered hypothesis assessment and results interpretation
  • Direct integration with Google Gemini and Anthropic Claude
  • Trade-off analysis and experiment planning tools
  • Export capabilities and auto-save functionality

🤖 AI Integration

This standalone edition connects directly to Google Gemini and Anthropic Claude APIs using your own API keys. All AI processing happens through your personal accounts - no middleman, no data sharing, complete privacy.

🔒 Privacy & Security

  • API keys stored securely in Chrome's encrypted storage
  • All statistical calculations run locally (no external calls)
  • AI analysis sent directly to your chosen provider
  • No data collection, logging, or third-party sharing
