ℹ️ About PM Tools

Chrome extension for A/B testing validation and analysis

🎯 Purpose

PM Tools is your statistical sidekick! We provide Product Managers with instant access to statistical validation and AI-powered insights for A/B testing experiments. Think of us as your friendly neighborhood data scientist who actually speaks PM.

🔑 Key Features

  • Experiment setup validation with statistical power analysis (no more guessing!)
  • Results interpretation with AI-generated insights (plain English, not statistics jargon)
  • Sample size and duration calculations (know before you go)
  • Trade-off analysis for different effect sizes (what if scenarios)
  • Export capabilities for sharing results (spreadsheet-ready data)

🔗 API Integration

This extension connects to a PM Tools API server that crunches the numbers and runs the AI analysis. Make sure your API server is running and accessible from your browser (it's the brains of the operation!).

🔒 Privacy

All data processing happens on your API server. This extension doesn't collect, store, or transmit any personal data to third parties. Your experiment data stays yours!
